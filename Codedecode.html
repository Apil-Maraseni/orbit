About

The Code-Decode Challenge is a competition designed to inspire tech enthusiasts to sharpen their mathematical problem-solving skills using programming. Participants will face real-world coding scenarios and mathematical challenges, offering a platform to showcase creativity and technical expertise. The competition is ideal for high school graduates and undergraduate students in fields like Engineering, IT, and Computer Science.

Structure

1. Activity: Mathematical Problem-Solving using code.

2. Team Size: Teams can have 1-3 participants.

3. Programming Languages: Participants may use any programming language of their choice (Python, C++, etc.).

4. Judging Criteria:

- Accuracy(50%): Correct solutions are crucial.

- Optimization(30%): Code should be efficient and optimized for speed.

- Code Readability(20%): Code should be well-documented and easy to follow.

5. Event Duration: 3.5 hours.

FAQs

FAQs:

1. Who can participate?
High school graduates and undergraduate students in Engineering, IT, and Computer Science are welcome to participate.

2. Can I participate alone?
Yes, participants can join solo or in teams of up to three members.

3. What programming languages can be used?
Any programming language like Python, C++, Java, etc. is allowed.

4. Will internet access be provided?
Yes, internet access will be provided to check syntax and libraries, but plagiarism is strictly prohibited.

5. How will the event be judged?
Judging will be based on the accuracy of solutions, code optimization, and readability.

Rules

1. Team Size: Teams can have 1-3 participants.

2. Duration: The challenge lasts 3.5 hours.

3. Submission: All solutions must be submitted within the time limit.

4. Language Choice: Any programming language can be used, but the final code should be readable and well-documented.

5. Plagiarism: Any form of plagiarism will result in disqualification.

6. Decision: The judges' decisions are final.
